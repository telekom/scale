# Design Tokens

Design Tokens sind ein zentraler Bestandteil des Scale Design Systems. Sie speichern Variablen wie Typografie, Farben, Abstände oder Radien. So entsteht ein skalierbares und visuell konsistentes System.

Die Tokens sind unabhängig von den Komponenten und liegen in CSS- und JavaScript-Formaten vor. So kannst du sie auch in deinen eigenen Komponenten verwenden.

## Mit Design Tokens arbeiten

Die Tokens sind als CSS-Variablen mit dem Präfix `scl` verfügbar.

Zusätzlich gibt es sie als benannten Export für die Nutzung in JavaScript (CSS-in-JS).

Um die Telekom Design Tokens zu nutzen, importiere sie aus dem Package `@telekom/scale-design-tokens`.

- `@telekom/scale-design-tokens/dist/design-tokens-telekom.css` für **CSS**
- `@telekom/scale-design-tokens/dist/design-tokens-telekom.js` für **JavaScript** (oder CSS-in-JS)

Den Quellcode findest du im Ordner `src/` verfügbar.

### Fonts formatieren

Um das Styling von Text zu erleichtern, gibt es einige CSS-Klassen:

| Anwendung  | CSS Klasse                   |
| ---------- | ---------------------------- |
| Heading 01 | .scl-font-variant-heading-1  |
| Heading 02 | .scl-font-variant-heading-2  |
| Heading 03 | .scl-font-variant-heading-3  |
| Heading 04 | .scl-font-variant-heading-4  |
| Heading 05 | .scl-font-variant-heading-5  |
| Heading 06 | .scl-font-variant-heading-6  |
| Body large | .scl-font-variant-body-large |
| Body long  | .scl-font-variant-body       |
| Body short | .scl-font-variant-body-short |

Nutze sie in HTML wie hier im Beispiel:

```html
<h1 class="scl-font-variant-heading-1">Typography</h1>
```
