## Heading Level

The default level for the heading (h1, h2, h3â€¦) inside the toggle button is `2`. So setting `heading-level` in `<scale-accordion>` will affect all `<scale-collapsible>` children automatically.

Adding `aria-level=""` to the heading slot on each item is not needed, but results in better accessibility in case JavaScript fails to load or execute.
