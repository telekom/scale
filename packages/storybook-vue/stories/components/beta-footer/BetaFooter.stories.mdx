## Data API (Backward compatibility)

> This scale-telekom-footer-data-back-compat component is only meant to make it easier to migrate from a [previous version of the Brand Footer](?path=/usage/components-footer--standard). If you do not need to migrate, please avoid it. Learn more about the [expected data shape](https://github.com/telekom/scale/blob/main/packages/storybook-vue/stories/components/brand-header-navigation/fixtures.js).
