## Using scale-app-shell over scale-header

Using scale-app-shell provides the page with the common region `content`, which helps you fulfill accessibility requirements. The scale-app-shell component includes both the Header and optionally the <a href="?path=/docs/components-footer--standard">Footer</a> component. However, there is a story <a href="?path=/docs/components-brand-header-navigation--standalone-header">demonstrating the standalone mode of the Header</a> for special cases.

## Dropdown Navigation Beta

You can find code examples where the Flyout Menu Beta component is implemented in the <a href="?path=/story/beta-components-flyout-menu--brand-header-primary-navigation">Primary navigation</a> and the <a href="?path=/story/beta-components-flyout-menu--brand-header-user-menu">User menu</a>.
