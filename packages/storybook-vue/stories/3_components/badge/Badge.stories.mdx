### Scoped CSS variables
```css
 :host {
  --color-badge:#e20074;
  --inner-width-small: 86px;
  --inner-height-small: 88px;
  --inner-width-big: 126.5px;
  --inner-height-big: 96px;
  --width-small: 120px;
  --height-small: 120px;
  --height-big: 160px;
  --width-big: 160px;
}
```
