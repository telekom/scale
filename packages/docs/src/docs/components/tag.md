# Tag

Tag component

selector: `t-tag`

## Default

```javascript
<t-tag>This is a tag</t-tag>
```

<div class="demo-container">
  <t-tag>This is a tag</t-tag>
</div>

## Pill

```javascript
<t-tag pill="true">Pill</t-tag>
```

<div class="demo-container">
 <t-tag pill="true">Pill</t-tag>
</div>

## With link

```javascript
<t-tag link="#">tag with link</t-tag>
```

<div class="demo-container">
 <t-tag link="#">tag with link</t-tag>
</div>

## Variants

Available `variants`:

- primary

```javascript
<t-tag variant="primary">primary</t-tag>
```

<div class="demo-container">
 <t-tag variant="primary">primary</t-tag>
</div>
