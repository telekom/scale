# Badge

Badge component

selector: `t-badge`

## Default

```javascript
<t-badge>default</t-badge>
```

<div class="demo-container">
  <t-badge>default</t-badge>
</div>

## Pill

```javascript
<t-badge pill="true">Pill</t-badge>
```

<div class="demo-container">
 <t-badge pill="true">Pill</t-badge>
</div>

## With link

```javascript
<t-badge link="#">badge with link</t-badge>
```

<div class="demo-container">
 <t-badge link="#">badge with link</t-badge>
</div>

## Variants

Available `variants`:

- dark
- light
- info
- warning
- danger
- success
- primary
- secondary

```javascript
<t-badge variant="primary">Primary</t-badge>
```

<div class="demo-container">
 <t-badge variant="primary">Primary</t-badge>
</div>
